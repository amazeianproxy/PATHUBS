# PATHUBS
A forum web application about connecting pet lovers, saving animals, and educating the masses about animals and pets.

# HOW TO SETUP
On terminaL:
1. cd server
2. npm install
3. cd ..
4. cd client
5. npm install

# HOW TO RUN
On terminal 1:
1. cd server
2. npm run dev

On terminal 2:
1. cd client
2. npm start
